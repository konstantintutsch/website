{% include heading.html level=3 text="1. Tracking" %}

This site does not use any tracking method.

No data will be shared with third parties.

{% include heading.html level=3 text="2. Data stored" %}

Metadata from HTTP requests is stored in the webserver's log.

This includes:

  • Requested file (e. g. that you visited /privacy)
  • User-Agent (often your browser and OS with corresponding version)
  • Client domain (this could reveal your ISP)
  • Referrer (e. g. you visited from search engine X)

{% include heading.html level=3 text="3. Hosting" %}

This website is hosted on a STRATO AG managed hosting server in Germany. STRATO provides a WebUI for easy access to the log data.

STRATO also retains the right to access and process all log data. See their privacy policy.
